# DataHub-itfest-2025
DataHub ВУЗов РК — ITFest 2025 — интерактивный сайт для школьников, помогающий выбрать университет. Для каждого вуза есть 3D-панорама кампуса, а встроенный ИИ-консультант отвечает на вопросы и рекомендует подходящий университет по интересам, специальностям и личным предпочтениям.
